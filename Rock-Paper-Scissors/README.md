# Rock Paper Scissors

## Overview
A terminal-based Rock Paper Scissors game where a Human plays against the Computer.
Tracks wins for both sides across multiple rounds until the player quits.

## Files
| File | Description |
|---|---|
| `rock_paper_scissors_game.py` | Complete game with win tracking and input validation |

## How to Run
```bash
python rock_paper_scissors_game.py
```
> No external libraries required. Python 3.x only.

## Rules
- 🪨 Rock beats Scissors
- ✂️ Scissors beats Paper
- 📄 Paper beats Rock
- Same choice = Draw

## Features
- ✅ Randomized computer choice each round
- ✅ Win counter for both Human and Computer
- ✅ Input validation with helpful error messages
- ✅ Quit anytime by pressing `Q`
- ✅ Final score summary on exit

## Concepts Used
- Procedural Programming
- Dictionaries for game rules mapping
- Loops and conditionals
- Random module
- String formatting (f-strings)
