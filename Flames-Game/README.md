# FLAMES Game

**Author:** Husnain Maroof  

A fun **Python terminal-based relationship prediction game** based on the classic **FLAMES algorithm**.  
The game compares two names, removes common characters, and determines a relationship result using the FLAMES elimination method.

---

## Overview

FLAMES is a popular game often played for entertainment to determine a fictional relationship between two people based on their names.

The acronym **FLAMES** represents:

| Letter | Meaning |
|------|------|
| F | Friend |
| L | Love |
| A | Affection |
| M | Marriage |
| E | Enemy |
| S | Siblings |

The program follows the traditional FLAMES process to determine the final relationship.
